Programs used in first lecture on network programming

csapp.{c,h}
	CS:APP wrappers.  Of particular interest are functions open_clientfd, and open_listenfd

hostinfo.c:
	Code demonstrating use of getaddrinfo and getnameinfo to retrieve DNS information

echoclient.c, echoserveri.c, echo.c:
	Echo client and server to demonstrate sockets code for client/server application
